# Guía de Contribución

¡Gracias por tu interés en contribuir a este proyecto!

## Cómo contribuir

1. Haz un fork del repositorio.
2. Crea una rama para tu feature o corrección (`git checkout -b mi-feature`).
3. Realiza tus cambios y asegúrate de que el código funcione correctamente.
4. Haz commit de tus cambios (`git commit -am 'Agrega nueva feature'`).
5. Haz push a tu rama (`git push origin mi-feature`).
6. Abre un Pull Request en GitHub y describe tus cambios.

## Requisitos de código

- Sigue la estructura y estilo de código existente (JavaScript y Java).
- Incluye comentarios claros y documentación si es necesario.
- Si modificas o agregas funciones, actualiza o crea pruebas si aplica.

## Reporte de errores

Si encuentras un error:
- Abre un Issue describiendo el problema, pasos para reproducirlo y, si es posible, adjunta capturas de pantalla o logs.

## Sugerencias y mejoras

- Si tienes ideas para mejorar el proyecto, abre un Issue o Pull Request y explícalo detalladamente.

## Contacto

Para dudas o ayuda, contacta a los mantenedores del repositorio.
